# Setup and Installation

## Install Nix

### WSL2 Single-User
```sh
% sh <(curl --proto '=https' --tlsv1.2 -L https://nixos.org/nix/install) --no-daemon
```

### WSL2 Multi-User (requires `systemd` enabled)
```sh
% sh <(curl --proto '=https' --tlsv1.2 -L https://nixos.org/nix/install) --no-daemon
```
